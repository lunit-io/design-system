# Lunit Design System Icons

> This is a collection of icons from the [Lunit Design System](https://www.figma.com/file/ygzUKBGrGxEYzkhUwoOC3U/Lunit-Design-System?node-id=4604%3A96681).

[![npm (scoped)](https://img.shields.io/npm/v/@lunit/design-system-icons)](https://www.npmjs.com/package/@lunit/design-system-icons)

## Installation

@mui/material v5 is required.

```shell
# with npm
npm install @lunit/design-system-icons

# with yarn
yarn add @lunit/design-system-icons
```

## Usage

Icons in this package is using [SvgIcon](https://mui.com/components/icons/#svgicon) wrapper.

```tsx
// Option 1
import SuccessIcon from "@lunit/design-system-icon/Success";
// Option 2
import { Success } from "@lunit/design-system-icon";

<Success />
// You can use any of SvgIcon props
<Success color="primary" />
<Success fontSize="large" />
// You can choose one of variants with `variant` props
<Success variant="filled" />
```

The safest for bundle size is Option 1, but some developers prefer Option 2. Make sure you follow the [minimizing bundle size guide](https://mui.com/material-ui/guides/minimizing-bundle-size/#option-2) before using the second approach.

### Variant

<Canvas>
  <Story name="variant">
    <SuccessIcon variant="outlined" />
    <SuccessIcon variant="filled" />

```tsx
React.ComponentProps<typeof SuccessIcon>["variant"]; // 'outlined' | 'filled'

SuccessIcon.variants; // ['outlined', 'filled']
```

The `outlined` is default if not set.

## Icons

<Grid container sx={{ color: "text.primary" }}>
  {Object.keys(Icons).map((key) => {
    const Icon = Icons[key];
    return (
      <>
        <Grid item xs={2} key={key}>
          <Typography>{key}</Typography>
        </Grid>
        <Grid item xs={10}>
          {Icon.variants
            .sort((a, b) => {
              if (a === "outlined") return -1;
              else if (b === "outlined") return 1;
              return 0;
            })
            .map((variant) => (
              <span title={`${Icon.displayName} ${variant}`}>
                <Icon key={Icon.displayName + variant} variant={variant} />
              </span>
            ))}
        </Grid>
      </>
    );
  })}
</Grid>
