# Lunit Design System Logo

> This is a collection of product logos from the [Lunit Design System](https://www.figma.com/file/tlomwt8dT0GWikKqzYH7TO/Lunit_Icons_v1.1.0?type=design&node-id=9065-96277&mode=design&t=SOt7e3SaZK7gN6Z2-11).

[![npm (scoped)](https://img.shields.io/npm/v/@lunit/design-system-logo)](https://www.npmjs.com/package/@lunit/design-system-logo)

## Installation

```shell
# with npm
npm install @lunit/design-system-logo

# with yarn
yarn add @lunit/design-system-logo
```

## Usage

```tsx
// Option 1 (Recommended)
import LunitLogo from "@lunit/design-system-logo/Lunit";
// Option 2
import { Lunit as LunitLogo } from "@lunit/design-system-logo";

<LunitLogo />
```

The safest for bundle size is Option 1, but some developers prefer Option 2. Make sure you follow the [minimizing bundle size guide](https://mui.com/material-ui/guides/minimizing-bundle-size/#option-2) before using the second approach.

The logo component itself is just a single SVG element. Any props you specify will also be passed to the SVG element.
